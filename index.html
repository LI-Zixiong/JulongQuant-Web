Regime 判断逻辑

市场状态判断器通过三个维度综合评分，识别当前处于牛市还是熊市，以及波动率的高低：

趋势 (Trend) — 基于市场指数在 20/60/120 三个时间窗口的风险调整动量，权重分别为 20%/45%/35%。使用因果归一化（仅用历史数据估计尺度），避免未来函数。

广度 (Breadth) — 衡量市场整体参与度，包括当日上涨股票比例、20日和60日正收益股票比例，权重分别为 15%/35%/50%。

波动率 (Volatility) — 20日年化波动率在过去 3 年中的分位数，通过 0.60/0.40 的迟滞阈值判定高/低波动状态。

综合评分 = 60% 趋势 + 40% 广度，再经迟滞机制（牛市需连续 5 天突破 0.50，熊市需连续 3 天跌破 -0.35）转换为稳定的牛/熊状态。最终将方向（牛/熊）与波动（高/低）组合为四种市场状态。

四种市场状态

低波动牛市Bull + LowVol — 趋势稳健，波动可控。维持仓位，逢回调可加仓。
高波动牛市Bull + HighVol — 趋势向上但波动加剧，警惕赶顶。考虑分批止盈，适度降低弹性。
低波动熊市Bear + LowVol — 阴跌格局，控制仓位。耐心等待右侧信号，不宜急于抄底。
高波动熊市Bear + HighVol — 波动剧烈且趋势向下。收紧仓位至最低，空仓睡觉战法。

数据来源

标的范围：中证 500 + 中证 1000 成分股。

更新频率：交易日每日更新。

时间跨度：2012 年至今。

巨龙量化

专注中证 500 + 1000 指数跟随策略的个人量化项目。通过系统化的因子研究和回测框架，在中证中小盘指数成分股中寻找超额收益机会。

市场状态识别

基于多维度趋势、广度与波动率信号，将市场划分为四种状态（低波动牛市 / 高波动牛市 / 低波动熊市 / 高波动熊市），辅助择时与仓位决策。本页面为该模块的可视化展示。
